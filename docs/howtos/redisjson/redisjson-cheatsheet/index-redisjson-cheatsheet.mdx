| Command	| Purpose        | Syntax	        |
| :--------------------- | :---------------------- | :----------------- |
| <button><b>[JSON.GET](https://oss.redis.com/redisjson/commands/#jsonget)</b></button> | Return the value at path in JSON serialized form | <code>JSON.GET &lt;key&gt</code>  |
| <button><b>[JSON.SET](https://oss.redis.com/redisjson/commands/#jsonset)</b></button> | Sets the JSON value at path in key  | <code>JSON.SET &lt;key&gt; &lt;path&gt; &lt;json&gt; [NX &#124; XX]</code> | 
| <button><b>[JSON.MGET](hTtps://oss.redis.com/redisjson/commands/#jsonmget)</b></button> | Returns the values at path from multiple key | <code>JSON.MGET &lt;key&gt; [key ...] &lt;path&gt;</code> |
| <button><b>[JSON.TYPE](https://oss.redis.com/redisjson/commands/#jsontype)</b></button> | Report the type of JSON value at path . | <code>JSON.TYPE &lt;key&gt [path]</code> | 
| <button><b>[JSON.NUMINCRBY](https://oss.redis.com/redisjson/commands/#jsonnumincrby)</b></button> |Increments the number value stored at path by number  | <code>JSON.NUMINCRBY &lt;key&gt; &lt;path&gt; &lt;number&gt;</code>  |
| <button><b>[JSON.NUMMULTBY](https://oss.redis.com/redisjson/commands/#jsonnummultby)</b></button> | Multiplies the number value stored at path by number | <code>JSON.NUMMULTBY &lt;key&gt &lt;path&gt &lt;number&gt</code> |
| <button><b>[JSON.STRAPPEND](https://oss.redis.com/redisjson/commands/#jsonstrappend)</b></button> | Append the json-string value(s) the string at path  | <code>JSON.STRAPPEND &lt;key&gt;  [path] &lt;json-string&gt;</code> |
| <button><b>[JSON.ARRAPPEND](https://oss.redis.com/redisjson/commands/#jsonarrappend)</b></button> |Append the json value(s) into the array at path after the last element in it | <code>JSON.ARRAPPEND &lt;key&gt; &lt;path&gt; &lt;json&gt;  [json ...]</code> |
| <button><b>[JSON.STRLEN](https://oss.redis.com/redisjson/commands/#jsonstrlen)</b></button>  | Report the length of the JSON String at path in key  | <code>JSON.STRLEN &lt;key&gt; [path]</code> |
| <button><b>[JSON.ARRLEN](https://oss.redis.com/redisjson/commands/#jsonarrlen)</b></button> | Report the length of the JSON Array at path in key | <code>JSON.ARRLEN &lt;key&gt; [path]</code> |
| <button><b>[JSON.ARRINSERT](https://oss.redis.com/redisjson/commands/#jsonarrinsert)</b></button> | Insert the json value(s) into the array at path before the index (shifts to the right) | <code>JSON.ARRINSERT &lt;key&gt; &lt;path&gt; &lt;index&gt; &lt;json&gt; [json ...]</code> |
| <button><b>[JSON.ARRINDEX](https://oss.redis.com/redisjson/commands/#jsonarrindex)</b></button> | Search for the first occurrence of a scalar JSON value in an array | <code>JSON.ARRINDEX &lt;key&gt; &lt;path&gt; &lt;json-scalar&gt; [start [stop]]</code> |
| <button><b>[JSON.ARRPOP](https://oss.redis.com/redisjson/commands/#jsonarrpop)</b></button> | Remove and return element from the index in the array | <code>JSON.ARRPOP &lt;key&gt;  [path [index]]</code> |
| <button><b>[JSON.ARRTRIM](https://oss.redis.com/redisjson/commands/#jsonarrtrim)</b></button> |Trim an array so that it contains only the specified inclusive range of elements | <code>JSON.ARRTRIM &lt;key&gt; &lt;path&gt; &lt;start&gt; &lt;stop&gt;</code> |
| <button><b>[JON.OBJKEYS](https://oss.redis.com/redisjson/commands/#jsonobjkeys)</b></button> | Return the keys in the object that's referenced by path | <code>JSON.OBJKEYS &lt;key&gt; [path]</code> |
| <button><b>[JSON.OBJLEN](https://oss.redis.com/redisjson/commands/#jsonobjlen)</b></button> |Report the number of keys in the JSON Object at path in key | <code>JSON.OBJLEN &lt;key&gt; [path] </code> |
| <button><b>[JSON.DEBUG](https://oss.redis.com/redisjson/commands/#jsondebug)</b></button> | Report information | <code>JSON.DEBUG &lt;subcommand & arguments&gt;</code> |
| <button><b>[JSON.RESP](https://oss.redis.com/redisjson/commands/#jsonresp)</b></button> | Return the JSON in key in Redis Serialization Protocol (RESP)| <code>JSON.RESP &lt;key&gt; [path]</code> ||
| <button><b>[JSON.FORGET](https://oss.redis.com/redisjson/commands/#jsonforget)</b></button> | An Alias for [JSON.DEL](https://oss.redis.com/redisjson/commands/#jsondel) | <code>JSON.DEL &lt;key&gt; [path]</code> |
